为什么有些 PDF 能打开但不能编辑？

PDF 可能设置了权限限制。某些文件允许查看和打印，但不允许编辑、注释、填写表单或签名。

本工具不会破解密码、移除权限或绕过保护。能打开查看，不代表一定能编辑。

本工具适合处理普通 PDF、自己创建的 PDF、扫描件、图片转 PDF、学习资料、简历、表格等你有权处理的文件。

    PDF 权限检测

    为什么有些 PDF 能打开但不能编辑？

    能打开查看不代表一定能编辑。PDF 可能允许查看和打印，但限制修改、复制、注释、填写表单、签名或页面提取。

    某些 PDF 可能包含数字签名，修改后可能导致签名失效。本工具不会处理 DRM、企业权限、在线授权或指定阅读器限制。

    上传 PDF 后会显示页数、预览状态和权限状态。

    本地加密说明

    PDF 加密码依赖 QPDF WebAssembly。文件和密码只在浏览器本地处理，不会上传，也不会保存。

    当前环境如果无法加载 qpdf-wasm，会提示按 README 部署或重新 vendor QPDF 模块。

    PDF 元数据清理

    本工具清理的是 PDF 文件内部元数据，例如标题、作者、关键词、创建工具、创建时间等。它不能修改 Windows / macOS 文件系统里的创建时间、访问时间、文件路径等操作系统属性。
